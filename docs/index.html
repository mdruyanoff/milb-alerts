How It Works

MiLB Game Alerts monitors live baseball games across all Minor League levels (AAA, AA, High-A, Single-A) and MLB Spring Training. When a game finishes, subscribers receive a text message with their tracked players' stats within 60 seconds.

Example alert (hitter):

Esteury Ruiz MLB/MIA @ HOU 1-3 | R, BB, SB Reply STOP to unsubscribe

Example alert (pitcher):

Germán Márquez MLB/SD vs CHC (W) 3.0 IP, 2 H, 0 R, 0 ER, 3 K, 0 BB, 0 HR Reply STOP to unsubscribe

Message frequency varies based on game schedules and the number of players you follow. During the baseball season (March–September), you may receive multiple messages per day.
