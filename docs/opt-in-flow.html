How End Users Provide Consent

1User texts JOIN to (310) 919-4684. The end user initiates contact by sending the keyword JOIN (or ALERTS or SUBSCRIBE) to the service number. This is the user's first action — no messages are sent by the service before this.

2User receives opt-in confirmation. The system replies with a confirmation message that includes the program name, message frequency disclosure, data rates notice, and opt-out instructions.

3User replies YES to confirm subscription. The user must reply YES to complete opt-in. No game alert messages are sent until this explicit YES reply is received and recorded.

Opt-Out Flow

Every alert message ends with: "Reply STOP to unsubscribe"

Text Message — (310) 919-4684

Example game alert with opt-out footer:

Germán Márquez MLB/SD vs CHC
(W) 3.0 IP, 2 H, 0 R, 0 ER, 3 K, 0 BB, 0 HR
Reply STOP to unsubscribe

End user opts out:

STOP

System confirms unsubscribe:

You have successfully been unsubscribed. You will not receive any more messages from this number. Reply START to resubscribe.
Key compliance points:
  • The end user always initiates by texting JOIN to (310) 919-4684
  • No messages are sent by the service before the user texts first
  • User must reply YES after the confirmation to complete opt-in
  • Users who reply NO or do not respond receive no further messages
  • Every alert message includes opt-out instructions
  • Replying STOP immediately unsubscribes at the carrier level
  • Consent is per-campaign, not transferable, and not a condition of any purchase
  • No mobile information is shared with third parties for marketing or promotional purposes
